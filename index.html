数学建模论文写作方法

数学建模论文的结构

首页：论文标题＋摘要＋关键词

一、问题重述

二、问题分析

三、模型假设

四、符号说明

七、模型的评价、改进与推广

八、参考文献

附录

〇、首页

论文标题

大部分的论文标题形式都是：

​ “基于……方法的……模型”

摘要

摘要是对论文内容的简要的陈述，不能超过一页。

摘要三要素：

解决了什么问题；应用了什么方法；得到了什么结论

摘要的结构：

开头段（必要）

针对问题一

针对问题二

……

结尾段（大部分论文没有）

开头段

需要充分概括论文内容，两到三句即可。

第一句可以简单交代一下题目的背景；

第二句需要交代论文所做的事情；

第三句可以说一下解决该问题的实际意义。

中间段

对每个问题分别进行叙述，一般包含三要素：解决了什么问题，应用了什么方法，得到了什么结论。很多论文不写解决了什么问题，直接交代方法。

在描述使用的模型时，要结合题目本身来叙述。

回答结果时，如果答案是具体的数值，直接回答即可。如果是开放的问题，在摘要中写主要的结论即可。

关键词

关键词一般放4-6个，可以放使用到的主要模型，也可以放论文中出现次数较多，能体现论文主要内容的词（一般不会放太多）。

一、问题重述

问题重述是整理原问题，用自己的语言叙述题目，可以加上题目背景，条件等。

二、问题分析

问题分析包括题目中的信息和条件，并对题目作整体的分析，确定用什么方法建立模型，一般每个问题单独分析一段，分析过程要简明扼要，不需要放结论。

建议在文字说明的同时用图形或者图表列出思维过程，会让思路显得更清晰。

问题分析一般不超过一页，题目较多时不超过两页。

三、模型假设

模型假设要合理，且是使用模型时所必需的假设。

常见的几种假设

  1. 题目明确给出的假设条件

  2. 生活中的小概率事件

  3. 仅考虑问题中的核心因素，不考虑次要因素的影响

    ​ 注意：过于简化模型会使得论文没有优势亮点

  4. 使用的模型要求的假设

  5. 对模型中的参数形式（或者分布）进行假设

    ​ 注意：能用数据验证这些假设更好

  6. 和题目联系很紧密的一些假设，主要是为了简化模型

四、符号说明

符号说明一般在建立模型之后写，把模型中用到的符号、参数整合在表中即可（模型中临时用到的变量可以不放）

在下文中首次出现的符号，也需要进行解释，不然会降低文章的可读性

五、模型的建立与求解

优化类问题模型的建立：目标函数＋约束条件

模型求解时的注意事项
  1. 国赛中经常出现优化类问题。如果使用到了启发式（智能优化）算法，需简要写明算法步骤，并结合具体问题来阐明计算的思路。
  2. 求解的结果要突出地展现出来，如果是开放类问题要清楚地阐明和解释，配上图表更好。

六、模型的分析与检验（可以只有检验，根据需要而定）

模型的分析与检验往往是比赛的加分项，虽然能与模型的建立与求解合并，但很多优秀的论文都会单独讨论这一部分。

模型的分析

模型的分析主要有两种：灵敏性分析，误差分析

灵敏性分析：

灵敏性分析研究的是系统或模型的输出结果对系统参数变化的灵敏程度。

通用的步骤是：控制其他参数不变的情况下，改变模型中某个重要参数的值，观察输出结果的变化情况。

误差分析：

误差分析是指分析模型中误差的来源，或者估算模型中存在的误差，一般用于预测问题或者数值计算类问题。

模型的检验

这里的检验不包含应在使用模型时做的检验，比如一致性检验等，而是对模型结果的检验，最常见的是稳定性检验。（与灵敏性检验比较类似）

七、模型的评价、改进与推广（三部分不一定都要写）

模型的评价：模型的优缺点（必要）

模型的改进：针对缺点的改进方法

模型的推广：将原题的要求进行拓展，进一步讨论模型的实用性和可行性

八、参考文献。

参考文献不仅能反应真实的科学依据，还能体现出严肃的科学态度和对他人成果的尊敬。所有引用他人或公开资料的成果必须按照科技论文的规范列出参考文献，并在正文引用处予以标注。

常见的三种参考文献的表达方式
  1. 书籍的表述方式为：[编号] 作者，书名，出版地：出版社，出版年月
  2. 期刊论文：[编号] 作者，论文名，杂志名，期卷号：起止页码，出版年
  3. 网上资源（例如数据库，政府报告）：[编号] 作者，资源标题，网址，访问时间

附录

附录内容应包括支撑材料的文件列表和所用到的源程序代码，也可以包含对问题的详细证明，较大的流程图等。

如果没有程序，可以在附录中说明“本论文没有用到程序”。

支撑材料包括用于支撑模型，结果，结论的所有必要材料，至少应包含源程序、查阅的数据资料、较大篇幅的中间结果的图表等。

支撑材料使用winRAR压缩在一个文件中，后缀为RAR或ZIP，不超过20M，将其文件列表放在论文附录中。

如果没有支撑材料，可以在附录中说明“本文没有支撑材料”。

圆锥曲线几何性质——抛物线（1-5）

抛物线的的几何性质（1-5）

抛物线定义

​ 给定一条直线L与直线外的一点S，找到一点使得该点与给定直线和与点的距离相等，所有这样的点的集合就是抛物线。给定的直线L称为准线，给定的点S称为焦点。

​ 抛物线的第一条性质指出了抛物线是轴对称图形，并且对称轴是过焦点且垂直准线的一条直线，这条性质这里不予证明。

在下文中，均记抛物线顶点为A，焦点为S，准线与抛物线对称轴交点为X.

性质2

PP’是抛物线上的一条弦，K是PP’延长线与准线的交点，则KS平分∠PSP’的外角

/抛物线性质2.png)

[证明]

图中过P, P’作准线的垂线，Q, Q’为垂足。根据抛物线的定义，有PQ = PS, P’Q’ = P’S.

因为PQ : P’Q’ = PK : P’K = PS : SK

所以根据角平分线定理，有SK平分∠PSP’的外角。

上面的证明用到了角平分线的一个定理，下面做简要介绍及证明：

（拓展）定理1

/角平分线性质_1.png)

在ΔABC中，CD为∠C的角平分线，则有CA : CB = DA : DB.

[证明]

图中DE, DE分别为过点D向CA, CB作的垂线。

因为CD是∠C的角平分线，因此有DE = DF。将DE, DF作为ΔACD和ΔBCD的高线，因为高线长度相等，所以ΔACD与ΔBCD面积之比为CA : BC.

而ΔACD与ΔBCD面积之比显然又等于AD : DB，因此有CA : CB = AD : BD.

这个命题的逆命题也是成立的，即若有CA : CB = AD : BD，则CD为∠C的角平分线，因为上面的推导过程完全可逆。

（拓展）定理2

/角平分线性质_2.png)

DA为∠C外角的角平分线，研究ΔDAB与ΔDAC的面积比值，可得到AB : AC = DB : DC, 证明方法与上面类似。

命题的逆命题同样也成立，上文抛物线的性质2的证明便是用到了这个逆命题。

性质3

/抛物线性质3.png)

PN垂直于抛物线对称轴，PQ垂直于准线，则有$PN^2=4AS\cdot AN$.

[证明]

所以：$PN^2=XN^2-NS^2=(2AS-NS)^2-NS^2$

整理得：$PN^2=4AS^2-4AS\cdot NS=4AS\cdot AN$

证毕。

当N为AS外侧时，结论仍成立，证明方法类似。

性质4

当N点与S点重合时，即过S作对称轴垂线交抛物线于P，由上面的结论得$PS=2AS$

性质5

/抛物线性质5.png)

过P作抛物线切线，交准线于点K，则有PS⊥KS

[证明]

设抛物线上另有一点P’，根据性质2，KS平分∠PSP’的外角。现在令P‘无限靠近P，则PP’也无限接近于过P点的切线，∠PSP’极限为0，∠PSK极限为$\frac {\pi}{2}$ .

由三角形全等性也易得P为∠SPQ的角平分线
